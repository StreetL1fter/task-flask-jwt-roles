# 🔐 Flask Auth & RBAC System

REST API на базе Flask с реализацией полноценной системы аутентификации (JWT) и авторизации (RBAC). Проект разработан с использованием современной архитектуры (Service Layer, Factory Pattern) и включает автоматическое тестирование.

## ✨ Реализованный функционал:

- ✅ **Аутентификация:** Регистрация и вход пользователей через JWT токены (PyJWT).
- ✅ **Авторизация (RBAC):** Гибкая система ролей и прав доступа (`@require_role`, `@require_permission`).
- ✅ **Защита маршрутов:** Декоратор `@require_auth` для проверки токенов.
- ✅ **Архитектура:** 
  - **Factory Pattern** для создания приложения (`create_app`).
  - **Blueprints** для модульности.
  - **Service Layer** для разделения бизнес-логики и маршрутов.
- ✅ **База данных:** SQLAlchemy ORM с SQLite (реальная база, не in-memory).
- ✅ **Безопасность:** 
  - Хеширование паролей (Werkzeug).
  - Черный список токенов (Logout).
  - Конфигурация через `.env`.
- ✅ **Тестирование:** Unit-тесты на успешную регистрацию и проверку дубликатов.
